MongoDb Install and Run Code
------------------------------------
docker pull mongo

docker run -d --name mongodb -p 27017:27017 -v mongodbdata:/data/db mongo

docker run -d --name mongodb --network my-network -p 27017:27017 -v mongodbdata:/data/db mongo

Project Docker Code
------------------------------------
docker build -t orders .

docker run -d --name orders --network my-network -p 5003:8080 -e ASPNETCORE_URLS=http://+:8080 orders
